# /trend <account> — TikTokジャンル流行観測

## 対象媒体
TikTok（ジャンル横断の広い観測。URL指定ではない）

## やること
1. サブエージェント `competitor-scanner` で TikTok のジャンル関連ハッシュタグ・おすすめを巡回
2. 今このジャンルで伸びている「形式」（例：Vログ系／作業風景＋テロップ）を集計
3. 前回観測からの変化を検出
4. `accounts/<account>/profile.md` の「今の流行り形式」を書き換える提案を出す

## 人の判断ポイント
profile.md を実際に更新するかは人が承認。

## 推奨頻度
週1程度の定点観測（競合解剖が都度なのに対し、流行りはゆっくり変わるため分離している）。
